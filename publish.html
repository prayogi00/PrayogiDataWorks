Title: Komparasi Algoritma Support Vector Machine dan CART untuk Klasifikasi Kualitas Udara DKI Jakarta

Authors: Rizki Prayogi Widartama, Maulana Fansyuri

Abstract:
Penelitian ini membahas klasifikasi kualitas udara di DKI Jakarta menggunakan algoritma Support Vector Machine (SVM) dan Classification and Regression Tree (CART). Berdasarkan hasil klasifikasi menggunakan data PM2.5 tahun 2021, model CART menunjukkan tingkat akurasi lebih tinggi (99,67%) dibandingkan SVM (95,05%).

Journal: OKTAL: Jurnal Ilmu Komputer dan Sains
Volume: 2, Number: 11, Year: 2023
